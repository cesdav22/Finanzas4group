Plan de pagos del programa fondo MiVivienda

El programa Fondo MiVivienda de Perú es una iniciativa gubernamental que tiene como objetivo ayudar a las personas de bajos y medianos ingresos a adquirir una vivienda propia. El programa funciona como una entidad financiera que otorga préstamos para la compra de viviendas a través de una red de bancos y financieras en todo el país.
